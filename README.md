# Vision du Cercle 100

Le Cercle 100 est un cadre collectif conçu pour analyser le monde actuel avant d’agir.

Sa vision repose sur :
- la compréhension des systèmes existants,
- la création de projets collectifs durables,
- une logique de transmission sur le long terme (100–200 ans).

Le Cercle 100 n’est pas un projet unique, mais un générateur de projets,
où chaque action s’inscrit dans une vision globale, collective et responsable.
